# NGO Event Explorer - KindredEvents

A modern React application for discovering and registering for social impact events.

## Features
- Dynamic event fetching from JSONPlaceholder API
- Real-time search by event name or location
- Filtering for upcoming and past events
- One-click event registration with feedback
- Registration counter
- Responsive UI

## Tech Stack
- React + Vite
- Tailwind CSS
- Framer Motion
- Lucide React

## Getting Started
1. Install dependencies: `npm install`
2. Run development server: `npm run dev`
3. Build for production: `npm run build`
